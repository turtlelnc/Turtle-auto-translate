# Turtle-auto-translate


## 混合语言 readme （mixed language readme）


中文简介（Chinese Introduction）：适用于chromium内核的的浏览器自动翻译，目前支持两种语言的自动整页单向翻译（英 to 中），如有异议以中文为主；

English Introduction（英文简介）：Browser automatic translation suitable for chromium core. Currently supports automatic full-page one-way translation in two languages ​​(English to Chinese). If there is any objection, Chinese will be the main language.
